# LinkedIn Account Creator API (Automation System)
This project is a FastAPI-based automation service that creates LinkedIn accounts using Selenium, GoLogin anti-detect browser profiles, and external verification services. It also handles email and SMS verification flows automatically.
The system simulates human-like behavior (typing delays, random actions, navigation patterns) and integrates AI-based CAPTCHA solving using OpenAI vision models.


## Features
Automated LinkedIn account creation flow
Human-like browser automation using Selenium
GoLogin integration for anti-detect profiles
Email verification via Mailgun webhook
SMS OTP handling via SMSPool API
AI-powered CAPTCHA solving using OpenAI vision model
Background task processing with FastAPI
Verification PIN storage system (file-based)
